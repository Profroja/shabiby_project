# PWA Icons Required

Please generate the following icon sizes and place them in this folder:

## Required Icons:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## How to Generate Icons:

### Option 1: Use PWA Builder (Recommended)
1. Visit: https://www.pwabuilder.com/
2. Upload a 512x512 logo/image with the Shabiby Cargo branding
3. Download the generated icon pack
4. Place all icons in this folder

### Option 2: Use Real Favicon Generator
1. Visit: https://realfavicongenerator.net/
2. Upload your logo
3. Download the generated icons
4. Place all icons in this folder

### Option 3: Use Favicon.io
1. Visit: https://favicon.io/
2. Use text to icon generator or upload an image
3. Download and extract
4. Place all icons in this folder

### Temporary Solution:
For testing purposes, you can use a simple colored square:
1. Create a 512x512 PNG with blue background (#4680ff)
2. Add white text "SC" (Shabiby Cargo)
3. Resize to all required sizes using an image editor

## Icon Design Guidelines:
- Use Shabiby Cargo brand colors
- Keep design simple and recognizable
- Ensure text is readable at small sizes (72x72)
- Use transparent or white background
- Include the 📦 emoji or truck icon if possible
